# termux-login-modifier

Packages you need:
     pkg install curl

• step1:
    curl -o https://raw.githubusercontent.com/alone1177/termux-login-modifier/main/login.txt

• step2:
    cat login.txt >> $HOME/../usr/etc/termux-login.sh

• step3:
    exit the termux. And open again.

• step4:
    Done !
